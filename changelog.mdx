## New features

- **Excursion statuses and types.** Define your own excursion statuses with custom names, icons, and hex colors, and manage reusable excursion types from your workspace settings. See [Excursions](/excursions).
- **Excursion visual identity.** Personalise each excursion with an icon and color so it's easier to recognise across lists and pages.
- **Approval flows timeline.** Review approvals with a step-by-step timeline that shows detailed descriptions for each stage of the process.
- **Public excursion website.** Configure a public-facing website for an excursion using a new two-column layout for settings, with redaction controls for what is visible.
- **Forms linked to excursions.** Create and link forms directly from an excursion, with support for special excursion checkbox fields (such as student and staff selection) in the form builder. See [Forms](/forms).
- **Form submissions list.** Browse and review form submissions in a dedicated list and grid view with column metadata, search, and sorting.
- **Parent permission emails.** Send permission requests to parents by email with tracked send attempts and a parent-facing access flow.
- **Post-excursion surveys.** Collect feedback from students and staff after an excursion with a built-in survey workflow.
- **PDF exports.** Generate PDFs for risk management plans and itineraries directly from an excursion.
- **Excursion participant status tracking.** Track each participant's status on an excursion individually.
- **Task risk summary.** View a risk summary on excursion task controls so you can see which risks each task addresses.
- **Workspace excursion settings.** Configure workspace-wide defaults that flow through to excursion planning and the weather modal.
- **Workspace excursion statuses.** Manage status options across the workspace with backfill support for existing data.
- **Staff compliance.** Track staff compliance information with default values and review it from staff records. See [Staff](/staff).
- **Student swimming ability.** Capture and validate students' swimming ability on their record. See [Students](/students).
- **Workspace member invite reminders.** Automatically remind invited members who haven't yet joined the workspace.
- **Sign-in security.** Turnstile CAPTCHA is now part of the auth flows to better protect your workspace.

## Updates

- **Forms tab on excursions.** The "Resources" tab on excursions has been renamed to **Forms** to better match how you use it.
- **Excursion staff list.** The staff members section on an excursion now supports search, filter, and sort.
- **Excursion details loading.** Loading states for excursion detail pages are smoother and clearer.
- **Form builder.** The builder now includes save functionality, an updated settings modal, and form type selection.
- **Excursion roles.** Excursion roles can now have a description to clarify responsibilities.
- **Approval flows.** Each step in the approval flow timeline now includes more detailed descriptions.
- **Data grids.** Rows now use dynamic heights, and cells support empty-state tooltips and improved display options.

## Fixes

- Fixed sign-out form submission handling.
- Fixed excursion status handling for several edge cases.
