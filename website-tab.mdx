The Website tab controls the public parent-facing page for an excursion. Use it to choose what public information appears, manage carousel images, publish or unpublish the page, and copy the public URL.

## What You Can See

- A Public website section.
- A public information alert explaining what is safe to show.
- Toggles for the content that appears on the public page.
- Carousel image upload and image management tools.
- A Publish public page section.
- Published or unpublished status.
- Public URL actions such as copy, preview, or create URL when available.

## Public Information Notice

The tab highlights that the public page should only include information suitable for families or the public audience. Sensitive records, internal notes, private student information, and restricted planning details should not be published.

## What Appears on the Public Page

You can choose which excursion details are shown publicly. Options may include:

- Overview or description.
- Location.
- Carousel.
- Learning focus.
- Plan for non-participating students.
- Primary excursion weather.
- Itinerary.
- Full itinerary detail.
- Itinerary times.
- Itinerary locations.

These toggles save changes as you use them.

## Carousel Images

The carousel images card lets you add images for the public page.

You may see:

- Add Images action.
- Drag-and-drop upload area.
- Validation messages if selected files cannot be used.
- Existing image thumbnails and names.
- Upload progress rows.
- Loading, error, and empty states.
- A note if you do not have document access or upload permission.

## Publishing

The publish section shows whether the parent page is published or unpublished. If you have permission, you can switch publishing on or off.

The public URL area may let you:

- Copy the URL.
- Open a preview.
- Create a URL if one has not been created yet.

## Common Workflows

### Prepare the public page

Choose the information that should appear, add carousel images if needed, then preview the page before publishing.

### Publish for families

Switch on publishing once the page content is ready and suitable for the intended audience.

### Share the page

Copy the public URL and share it through the appropriate school communication channel.

## Tips

- Preview before sharing the public URL.
- Only enable sections that are complete and appropriate for families.
- Keep sensitive planning details in internal tabs, not on the public page.
