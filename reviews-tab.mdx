The Reviews tab manages the approval process for an excursion. Use it to submit an excursion for review, follow approval progress, complete review actions, and view the approval audit trail.

## What You Can See

- Loading and error states while review information is being prepared.
- A Submit for review area when the excursion can be submitted.
- Approval flow selection and description.
- A message when no active approval flows are available.
- Review status summary cards.
- A step-by-step approval timeline.
- Cards for each approval step.
- Reviewer actions such as approve, send back, or return to organiser when permitted.
- A comment box for review decisions.
- An audit trail showing review activity.
- A reassignment modal when approvers can be changed.

## Submit for Review

When available, choose an approval flow and submit the excursion for review. The selected flow controls the steps and approvers used for the review.

If no active approval flows are available, the tab shows an alert so you know the excursion cannot be submitted through a flow yet.

## Review Status

The current review state may show:

- Approval flow.
- Overall status.
- Current step.
- Last updated time.

This gives organisers and reviewers a quick summary of where the excursion sits in the approval process.

## Approval Steps

Each step can show:

- Step status.
- Description.
- Assigned approver.
- Eligible approvers.
- Who actioned the step and when.
- Manage options such as reassignment when allowed.

## Reviewer Actions

When you are the current reviewer, you may be able to:

- Add a comment.
- Send the excursion back one step.
- Return the excursion to the organiser.
- Approve the current step.

If another person needs to act, the tab shows a waiting message instead.

## Audit Trail

The audit trail records review-related actions. If no review actions have occurred yet, the tab shows an empty state.

## Common Workflows

### Submit an excursion

Choose the correct approval flow, check the description, then submit the excursion for review.

### Complete a review step

Read the excursion details, add a comment if helpful, then approve or return the excursion depending on what is needed.

### Reassign an approver

Use the manage option on a step when responsibility needs to move to another eligible person.

## Tips

- Review key planning tabs before submitting for approval.
- Use comments to explain review decisions clearly.
- Check the audit trail when you need to understand previous review actions.
