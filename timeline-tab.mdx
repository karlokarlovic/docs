The Timeline tab is where you plan the schedule for an excursion. Use it to create itinerary events, view the excursion in calendar formats, and check map or weather information when available.

## What You Can See

- A Timeline heading with actions for map, weather, and new events.
- A calendar view for excursion itinerary events.
- Day, week, month, year, and agenda-style views.
- Event details shown through calendar interactions.
- Loading, error, and empty states.
- A delete confirmation when removing itinerary events.

## Main Actions

### View Map

Use View Map to see itinerary events with location information in a map-focused view. This helps staff understand travel flow, venue order, and where events are happening.

### View Weather

Use View Weather to review weather information for itinerary events where supported. This is especially useful for outdoor activities or transport planning.

### New Event

Use New Event to add an itinerary event. Depending on your permissions, this action may be disabled in read-only situations.

An itinerary event can represent activities, travel, meals, accommodation, briefings, or other scheduled parts of the excursion.

## Calendar Features

- Switch between day, week, month, year, and agenda views.
- Select a time range to create an event when editing is allowed.
- Open an existing event to review or update it.
- Edit event details in the event form.
- Delete an event after confirming the action.

## Empty and Loading States

If no itinerary events have been added, the tab shows an empty message. If events are still loading, the tab shows a loading state. If something goes wrong, an error message appears.

## Common Workflows

### Build the excursion schedule

Add each major part of the day as an event, including departure, arrival, activities, meals, transitions, and return travel.

### Check timing gaps

Use calendar views to spot missing times, overlaps, or parts of the itinerary that may need more detail.

### Review travel and conditions

Use map and weather views to support planning decisions, especially for outdoor or multi-location excursions.

## Tips

- Add enough detail for staff to follow the day without needing a separate schedule.
- Review the timeline before finalising transport or publishing parent-facing information.
- Use the agenda view when you want a simple list of itinerary events.
