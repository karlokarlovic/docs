---

The Documents tab stores files and folders connected to an excursion. Use it to organise planning documents, attachments, generated files, and supporting material in one place.

## What You Can See

- A Documents heading and introduction.
- A folder and file tree.
- Folder expand and collapse controls.
- Actions to add folders or upload files when you have permission.
- Actions to open, delete, or manage files and folders where available.
- A properties column showing excursion status, creator, creation date, and location.
- Loading, error, permission, and empty states.

## Main Actions

### Add a Folder

Create root folders or subfolders to organise documents by topic, such as permissions, transport, risk, medical, or finance.

### Upload a File

Upload files into the appropriate folder. Files can include planning notes, venue information, forms, images, signed documents, or other excursion material.

### Open a File

Open files from the document tree when you need to view or download them.

### Delete a File or Folder

Use delete actions to remove items that are no longer needed. Deletion may require confirmation and may depend on your permissions.

## Properties

The right column repeats key excursion information while you work with documents. This helps you confirm that files are being added to the correct excursion.

## Common Workflows

### Organise planning files

Create folders for major planning areas, then upload related files into each folder.

### Review supporting documents

Open files from the tree when checking excursion readiness, approval evidence, or parent-facing material.

### Clean up old material

Delete duplicate or outdated files when you are sure they are no longer needed.

## Tips

- Use clear folder names so staff can find documents quickly.
- Keep final versions separate from working drafts where possible.
- Check permissions if upload or delete actions are not available.
