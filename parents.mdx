The Parents page helps you manage parent, carer, and guardian records. These records support communication, permissions, student relationships, and excursion planning.

## What You Can Do

- Browse parent and carer records.
- Create a new parent record.
- Edit parent details.
- Import parent records when adding many contacts at once.
- Manage relationships between parents or carers and students.
- Keep family contact information available for communication and permissions.
- Review relationship links before sending information to families.

## Common Workflows

### Add a parent or carer

Create a parent record with the contact details staff need for communication and permissions.

### Import family contacts

Use import when you have a larger set of parent or carer records to add.

### Connect a parent to a student

Open relationship management for the parent and link the relevant student records.

## Tips

- Keep contact details accurate before sending forms, permissions, or excursion updates.
- Check student relationships carefully when family circumstances change.
- Use clear names and details so staff can distinguish contacts with similar names.
