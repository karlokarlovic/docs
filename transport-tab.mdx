The Transport tab manages transport-related itinerary events for an excursion. Use it to record buses, flights, trains, accommodation movements, and other vendor-based transport arrangements.

## What You Can See

- A Transport heading with a count.
- A description of transport events.
- A New transport event action.
- A transport table with filters and row actions.
- Loading, error, empty, and filtered empty states.
- A create or edit modal for transport events.
- A delete confirmation when removing a transport event.

## Transport Table

The table can show:

- Title, with a short description snippet.
- Vendor.
- Type, such as bus, flight, hotel, or train.
- Location.
- Start and end times.
- Alerts, such as missing start or end details.
- Actions, such as edit or delete.

Rows may open the edit form when clicked if you have permission.

## Main Actions

### New Transport Event

Use New transport event to add a transport-related schedule item. This can include travel, accommodation check-in, transfers, or other vendor-managed movements.

### Edit Transport Event

Open an event to update its title, vendor, type, location, time, description, or related details.

### Delete Transport Event

Use the delete action when a transport event is no longer needed. You may need to confirm the deletion.

## Alerts and Filters

The table can highlight transport events with missing start or end information. Use filters to find incomplete records that need follow-up.

## Common Workflows

### Add transport arrangements

Create transport events for departure, transfers, accommodation movements, and return travel.

### Check missing details

Use alerts to find transport records without complete timing information.

### Confirm vendor information

Review the vendor and location columns before finalising itinerary or parent-facing details.

## Tips

- Keep transport times current because they affect the excursion timeline.
- Use clear titles such as "Bus to museum" or "Return flight".
- Review transport alerts before generating or sharing planning documents.
