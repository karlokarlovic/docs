---
title: "Tasks Tab"
---

The Tasks tab shows tasks linked to the excursion. Use it to plan follow-up work, assign responsibilities, track due dates, and keep planning actions connected to the excursion record.

## What You Can See

- A Tasks heading with a count of linked tasks.
- A New Task action when you have permission to create tasks.
- A task table focused on this excursion.
- Loading and empty states.
- Task editing through a side panel or overlay.
- Row actions for editing or deleting tasks when allowed.

## Task Table

The table may include:

- Title, describing the task.
- Status, showing progress.
- Priority, showing importance.
- Due, showing the due date.
- Assigned User, showing who owns the task.
- Excursion Role, showing the role connected to the task.
- Updated, showing when the task last changed.

The table supports standard browsing tools such as sorting, filtering, and column visibility where available.

## Main Actions

### New Task

Use New Task to create a task already linked to the excursion. This is useful for actions such as confirming transport, checking forms, assigning supervision, or following up with families.

### Edit Task

Open a task from the table or row actions to update its title, status, assignment, priority, due date, or other details.

### Delete Task

Use the row action menu to delete a task when it is no longer needed and you have permission to remove it.

## Common Workflows

### Create planning actions

Add tasks for anything that needs a clear owner or deadline during excursion planning.

### Review outstanding work

Filter or sort the table by status, priority, assignee, or due date to see what still needs attention.

### Open a shared task

If a task link includes a task reference, the excursion can open directly to that task so you can review or update it quickly.

## Tips

- Use specific task titles so other staff know exactly what needs to happen.
- Assign tasks to a person or role when responsibility should be clear.
- Review tasks before submitting an excursion for approval.
