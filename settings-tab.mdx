The Settings tab manages visual settings for an excursion. Use it to choose the icon and colour that help identify the excursion across the workspace.

## What You Can See

- A Settings heading.
- A loading message if settings are not ready yet.
- An Icon and colour section.
- An icon and colour picker.
- Field-level error messages when something needs attention.
- A general error alert if saving fails.
- A Save changes button when edits are available.

## Icon and Colour

The icon and colour help staff recognise the excursion in lists, headers, and other places where the excursion is shown.

You can:

- Choose an icon.
- Choose a colour.
- Clear the selection when supported.
- Save changes when the selection has been updated.

## Permissions

If you do not have permission to manage excursion settings, editing or saving may be unavailable. You can still view the current visual identity where it is shown.

## Common Workflows

### Set the excursion identity

Choose an icon and colour that make the excursion easy to recognise, then save the changes.

### Update an existing identity

Change the icon or colour if the current choice no longer fits the excursion or is too similar to another record.

### Leave the default appearance

If no special visual identity is needed, leave the settings unchanged.

## Tips

- Use colours and icons consistently for similar types of excursions where that helps staff.
- Save changes before leaving the tab.
- Choose a simple icon that remains easy to recognise at small sizes.
