The Tasks page helps staff keep track of work that needs to be completed across the workspace. Tasks can be used for planning, follow-up, review, and day-to-day coordination.

## What You Can Do

- Browse workspace tasks in one list.
- Create a new task for yourself or another staff member.
- Open a task to review its details.
- Edit task information when plans change.
- Track work related to excursions or general workspace activity.
- Use task status to understand what is still open and what has been completed.
- Open a task directly from a shared link when available.

## Common Workflows

### Create a planning reminder

Add a task for an action that needs follow-up, such as confirming transport, checking medical notes, or preparing parent communication.

### Review your outstanding work

Open the Tasks page to check what needs attention and update tasks as work progresses.

### Follow a shared task link

If someone shares a task link with you, open it to jump straight to the task details.

## Tips

- Use clear task names so other staff can quickly understand the action required.
- Keep task details current if responsibility, timing, or status changes.
- Review tasks regularly during excursion planning so small actions do not get missed.
