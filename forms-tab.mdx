The Forms tab manages forms connected to an excursion. Use it to connect existing workspace forms, build new forms, send permission emails, and collect post-excursion feedback.

## What You Can See

- A Forms heading with a count.
- Links and actions for workspace forms, permission emails, surveys, building forms, and connecting forms.
- A table of connected forms.
- A connect existing form modal.
- Empty states and hints when forms are not yet connected.
- Guidance when a published permission form is needed before permission emails can be sent.

## Main Actions

### Workspace Forms

Use this link to open the broader workspace forms area, where reusable forms can be managed.

### Send Parent Permission Emails

Use this action to send permission emails to families when the excursion has a suitable published permission form.

### Send Post-Excursion Surveys

Use this action to send survey forms after the excursion when an appropriate survey form is connected.

### Build New Form

Use Build new form to create a form for the excursion when you need a new permission form, survey, or information collection form.

### Connect Existing Form

Use Connect existing form to choose a form already available in the workspace and link it to the excursion.

### Unlink Form

Use Unlink to remove a form from the excursion when it should no longer be connected.

## Forms Table

The table can show:

- Title and description.
- Type, such as permission or survey.
- Published status.
- Last updated time.
- Open link.
- Unlink action.

## Common Workflows

### Prepare permission collection

Connect or build a permission form, publish it, then use the permission email action when ready.

### Add a survey

Build or connect a survey form, then use it for post-excursion feedback.

### Review connected forms

Check the table before communicating with families to make sure the right forms are connected and published.

## Tips

- Permission emails usually require a published permission form.
- Use clear form titles so staff can identify the right form quickly.
- Open a form before sending it if you need to check the wording or questions.
